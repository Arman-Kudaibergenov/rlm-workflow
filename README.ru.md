# RLM Workflow для Claude Code

**[Read in English](README.md)**

## Проблема

Каждый раз когда контекстное окно Claude заполняется и срабатывает `/compact`, вы теряете всё: принятые решения, обсуждённую архитектуру, незавершённые задачи. Следующая сессия — начинаете с нуля.

**RLM решает это**, давая Claude персистентное внешнее хранилище памяти, которое сохраняется между очистками контекста.

## Что такое RLM

[RLM-Toolkit](https://github.com/DmitrL-dev/AISecurity/tree/main/rlm-toolkit) — open-source слой памяти для LLM-агентов, разработанный [Дмитрием Лабинцевым (DmitrL-dev)](https://github.com/DmitrL-dev). Реализует иерархическую память с отслеживанием причинно-следственных решений, семантическим поиском и корпоративным управлением контекстом.

Этот репозиторий документирует наш **кастомизированный рабочий процесс** на базе RLM-Toolkit для ежедневной разработки с Claude Code.

## Быстрый старт (Docker)

```bash
# Запустить RLM-сервер
docker run -d \
  --name rlm \
  -p 8200:8200 \
  -v rlm-data:/data \
  ghcr.io/YOUR_USERNAME/rlm:latest

# Добавить в конфиг Claude Code MCP (~/.claude/mcp.json):
# см. docs/ru/установка.md для полной конфигурации
```

Или через Docker Compose:
```bash
curl -O https://raw.githubusercontent.com/YOUR_USERNAME/rlm/main/docker/docker-compose.yml
cp docker/.env.example .env
# Отредактируйте .env под свои настройки
docker compose up -d
```

## Что кастомизировано

Наш workflow расширяет оригинальный RLM-Toolkit:

| Функция | Оригинал | Наш workflow |
|---------|----------|--------------|
| Непрерывность сессий | Вручную | Автоматически через pre-compact hook |
| Мониторинг контекста | Нет | Авто-сохранение при 65% заполнения |
| Память команды агентов | Нет | Общий RLM для мульти-агентных команд |
| Отслеживание задач | Нет | PENDING-факты с привязкой к проекту |
| Автозахват | Нет | Автоматическое логирование операций |

Полный список: [CHANGELOG.md](CHANGELOG.md)

## Варианты LLM-бэкенда

RLM-Toolkit поддерживает разные провайдеры для семантического поиска и эмбеддингов:

| Вариант | Требования | Сложность настройки |
|---------|-----------|---------------------|
| OpenAI API | Нет (облако) | Низкая — только API-ключ |
| Ollama + nomic-embed | Только CPU | Средняя |
| Ollama + qwen3:8b | Рекомендуется GPU | Средняя |
| FastEmbed (встроенный) | Только CPU | Нулевая |
